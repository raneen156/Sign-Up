# Responsive Sign Up Form

واجهة تسجيل حساب (Sign Up Form) بسيطة وعصرية تم تطويرها باستخدام:

- HTML5
- CSS3
- JavaScript
- Font Awesome

تتميز بتصميم نظيف ومتجاوب (Responsive) يعمل على مختلف أحجام الشاشات.

---

## Preview

واجهة تسجيل حديثة تحتوي على:

- حقل الاسم
- البريد الإلكتروني
- كلمة المرور
- تأكيد كلمة المرور
- زر تسجيل الدخول
- روابط تسجيل عبر مواقع التواصل
